# AWS SAA-C03 Exam Simulator

https://chalut01.github.io/aws-saa-c03-exam/

โปรเจคนี้เป็นเว็บแอปแบบ client-side สำหรับจำลองข้อสอบ AWS Certified Solutions Architect - Associate (SAA-C03) โดยใช้ไฟล์ JSON ของคำถามเป็นข้อมูล

## สิ่งที่โปรเจคทำได้

- โหลดไฟล์คำถามจากไฟล์ JSON ผ่านการอัปโหลดในเบราว์เซอร์
- สุ่มลำดับข้อสอบก่อนเริ่มทำได้
- รองรับคำถามแบบ
  - `single` (เลือกคำตอบเดียว)
  - `multiple` (เลือกหลายคำตอบ)
- แสดงผลคะแนนทันที
- แสดงคำอธิบายเฉลยหลังตรวจคำตอบ
- แสดงสรุปคะแนนหลังทำข้อสอบครบ

## ไฟล์สำคัญ

- `index.html` - หน้าเว็บหลักของ Exam Simulator
- `aws-saa-c03-full-exam.json` - ตัวอย่างไฟล์คำถาม JSON
- `aws-saa-c03-question.txt` / `aws-saa-c03-answer.txt` - ไฟล์ที่รวมคำถามและคำตอบจากชุดข้อสอบ

## วิธีใช้งาน

1. เปิดไฟล์ `index.html` ในเบราว์เซอร์ (สามารถดับเบิลคลิกหรือเปิดผ่าน Live Server ได้)
2. คลิกปุ่มอัปโหลดแล้วเลือกไฟล์ JSON ของคำถาม เช่น `aws-saa-c03-full-exam.json`
3. กดเริ่มทำข้อสอบ
4. เลือกคำตอบ แล้วกด `ตรวจคำตอบ`
5. ดูผลคะแนนและคำอธิบายเฉลย
6. กด `ข้อถัดไป` เพื่อไปยังข้อถัดไปจนจบ

## รูปแบบไฟล์ JSON ที่รองรับ

ไฟล์ JSON ต้องเป็นอาร์เรย์ของวัตถุคำถาม และแต่ละคำถามต้องมีฟิลด์ต่อไปนี้:

- `id` - หมายเลขหรือรหัสคำถาม
- `topic` - หมวดหมู่หัวข้อ (ข้อมูลสำหรับจัดกลุ่ม)
- `domain` - ชื่อโดเมนหรือหมวดเนื้อหา
- `type` - ประเภทคำถาม (`single` หรือ `multiple`)
- `question` - ข้อความคำถาม
- `options` - รายการตัวเลือก
- `correctAnswers` - อาร์เรย์รหัสตัวเลือกที่ถูกต้อง
- `explanation` - ข้อความคำอธิบายเฉลย
- `keywords` - คีย์เวิร์ดเสริม (ไม่จำเป็นต้องใช้งาน)

ตัวอย่างโครงสร้างคำถาม:

```json
{
  "id": 1,
  "topic": "Topic 1",
  "domain": "Storage",
  "type": "single",
  "question": "คำถาม...",
  "options": [
    { "id": "A", "text": "ตัวเลือก A" },
    { "id": "B", "text": "ตัวเลือก B" }
  ],
  "correctAnswers": ["A"],
  "explanation": "คำอธิบายเฉลย...",
  "keywords": []
}
```

สำหรับคำถามประเภท `multiple` ให้ใส่ `correctAnswers` เป็นอาร์เรย์ของตัวเลือกหลายตัว เช่น `["A","C"]`

## หมายเหตุ

- แอปทำงานทั้งหมดบนฝั่งไคลเอ็นต์ ไม่ต้องตั้งค่าเซิร์ฟเวอร์
- สามารถใช้ไฟล์ JSON เองได้ หากมีรูปแบบตรงตามที่ระบุ
- หากมีข้อผิดพลาดในการอ่านไฟล์ JSON แอปจะแจ้งเตือนให้ตรวจสอบรูปแบบไฟล์
